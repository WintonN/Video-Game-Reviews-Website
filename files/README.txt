# Video Game Reviews

Video Game Reviews is a website project by Winston Niogret (Student ID: 45321407) that aims to provide unbiased and honest reviews of video games.

# Directory

The root folder contains all HTML files which correspond to each of the individual website pages.
The css folder, located in the root folder, contains all the css files used for the website.
The js folder, located in the root folder, contains all the javascript files for various functions in the website.
The images folder, located in the root folder, contains all the images used in the website.
The files folder, located in the root folder, contains all other various files, such as the README, website poster, and reference list.
